# `@dorai-ui/alert`

The accessible, unstyled fully functional alert component from @dorai-ui.

## Installation

```sh
npm i @dorai-ui/alert
```

or

```sh
yarn add @dorai-ui/alert
```

## Example Usage

```
import { Alert } from '@dorai-ui/alert'

function AlertComponent()  {
  return (
    <Alert>
      <p>Alert component</>
    </Alert>
  )
}
```
