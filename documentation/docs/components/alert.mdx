# Alert

> **Alert component displays a short important message to the user and adheres to [WAI-ARIA Alert pattern](https://www.w3.org/TR/wai-aria-practices-1.2/#alert).**

### Core Features

- Out of the box Accessible

### Installation

```sh
npm i @dorai-ui/alert

or

yarn add @dorai-ui/alert
```

### API

#### Components Composed

```jsx
import { Alert } from '@dorai-ui/alert'

const AlertComp = () => <Alert />
```

#### `Alert`

The main Alert component.

| Props | &nbsp; Default &nbsp; | &nbsp;Description                                                             |
| ----- | --------------------- | ----------------------------------------------------------------------------- |
| `as`  | 'div'                 | This grants the ability to change the element this component should render as |
